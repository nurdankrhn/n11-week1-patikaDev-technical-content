# Prime Number Check with Recursion

Bu program, kullanıcıdan alınan bir sayının **asal** olup olmadığını **recursive** (özyinelemeli) metot kullanarak kontrol eder.

## Asal Sayı Nedir?
Bir sayının yalnızca 1 ve kendisiyle tam bölünebiliyorsa, o sayı asal sayıdır. Örneğin:
- `2` → asal sayıdır.
- `4` → asal değildir çünkü 2'ye bölünebilir.

## Kullanılan Konular
- Java
- Scanner
- Recursive method (Özyinelemeli fonksiyon)
- if koşulu
