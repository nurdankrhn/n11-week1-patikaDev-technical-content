# Palindrome Number Checker

Bu program, bir sayının palindrom sayı olup olmadığını kontrol eder.

## Palindrom Sayı Nedir?
Palindromik sayı, tersine okunduğunda da aynı kalan sayılardır. Örneğin:
- `121` → Tersine de `121` olur.
- `123` → Tersine `321` olur, bu nedenle palindrom değildir.

## Kullanılan Konular
- Java
- Scanner
- while döngüsü
- if koşulu
