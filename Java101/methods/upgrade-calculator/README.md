# Gelişmiş Hesap Makinesi

Bu program, kullanıcıya çeşitli hesaplama işlemleri sunar ve menüde seçilen işlemi gerçekleştirir. 

## İşlemler
1. **Toplama İşlemi**
2. **Çıkarma İşlemi**
3. **Çarpma İşlemi**
4. **Bölme İşlemi**
5. **Üslü Sayı Hesaplama**
6. **Faktoriyel Hesaplama**
7. **Mod Alma**
8. **Dikdörtgen Alan ve Çevre Hesaplama**

## Kullanılan Konular
- Java
- while döngüsü
- switch-case yapısı
- method kullanımı
