# Recursive Power Calculator

Bu program, kullanıcıdan alınan taban ve üs değerleri ile üslü sayıyı **recursive** (özyinelemeli) metot kullanarak hesaplar.

## Özellikler
- Kullanıcıdan taban ve üs değeri alınır.
- Üslü sayı hesaplaması **recursive** metotla yapılır.
- Herhangi bir sayının 0. kuvveti 1 olarak kabul edilir.

## Kullanılan Konular
- Java
- Scanner
- Recursive method (Özyinelemeli fonksiyon)

javac Main.java
