# Palindromik Kelime Kontrolü Programı

Bu program, kullanıcı tarafından girilen bir kelimenin palindromik olup olmadığını kontrol eder. Palindromik kelime, tersten okunduğunda da aynı kelimeyi veren bir kelimedir.

## Kullanılan Yöntemler:
1. **isPalindrome**: Birinci yöntemle, iki indeks kullanarak (başlangıç ve bitiş) her iki yandan karşılaştırma yaparak palindrom olup olmadığını kontrol eder.
2. **isPalindrome2**: İkinci yöntemde, string'in tersini alıp orijinal string ile karşılaştırır.

## Kullanılan Konular:
- Java
- String manipülasyonu
- Döngüler ve koşullar
