# Yıldızlarla B Harfi Yazdırma Programı

Bu program, çok boyutlu diziler kullanarak ekrana "B" harfini yıldızlarla yazdırır.

## Özellikler
- 6x4 boyutunda bir dizi oluşturulur ve her eleman için uygun koşullara göre yıldız (*) yerleştirilir.
- Program, "B" harfini oluşturacak şekilde yıldızları yerleştirir ve sonrasında ekrana basar.

## Kullanılan Konular
- Java
- Çok Boyutlu Diziler
- Döngüler

## Çalıştırma
```bash
javac Main.java
java Main
