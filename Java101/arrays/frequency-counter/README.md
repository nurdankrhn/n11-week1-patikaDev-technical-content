# Dizideki Elemanların Frekanslarını Hesaplayan Program

Bu program, kullanıcıdan alınan bir dizi içindeki her elemanın kaç kez tekrar ettiğini (frekansını) hesaplar.

## Özellikler
- Kullanıcıdan dizinin boyutu ve elemanları alınır.
- Her elemanın kaç kez tekrar ettiği, bir `HashMap` kullanılarak hesaplanır.
- Sonuçlar ekrana yazdırılır.

## Kullanılan Konular
- Java
- HashMap
- Döngüler
- Scanner

## Çalıştırma
```bash
javac Main.java
java Main
