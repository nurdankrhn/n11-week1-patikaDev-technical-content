# Dizideki Elemanların Max ve Min Değerlerini Bulan Program

Bu program, verilen dizideki elemanlar arasından, kullanıcı tarafından girilen sayıya en yakın küçük ve büyük sayıyı bulur.

## Özellikler
- Kullanıcıdan alınan sayıya göre dizinin en yakın küçük ve büyük sayıları hesaplanır.
- Kullanıcı girdisi ile dizideki sayıların karşılaştırılması yapılır.

## Kullanılan Konular
- Java
- Döngüler
- Diziler
- Scanner

## Çalıştırma
```bash
javac Main.java
java Main
