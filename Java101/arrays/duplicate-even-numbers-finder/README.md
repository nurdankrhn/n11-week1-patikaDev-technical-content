# Duplicate Even Numbers Finder

Bu program, bir sayı dizisindeki tekrar eden çift sayıları bulur ve ekrana yazdırır.

## Özellikler
- Kullanıcıdan alınan diziyi kontrol eder.
- Tekrar eden çift sayıları tespit eder ve bu sayıları ekrana yazdırır.

## Kullanılan Konular
- Java
- `HashSet` kullanımı
- Döngüler
- `if` koşulu

## Çalıştırma
```bash
javac Main.java
java Main
