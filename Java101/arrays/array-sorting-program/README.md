# Diziyi Küçükten Büyüğe Sıralayan Program

Bu program, kullanıcıdan alınan dizinin boyutu ve elemanları ile diziyi küçükten büyüğe sıralar.

## Özellikler
- Kullanıcıdan dizinin boyutu alınır.
- Kullanıcıdan her bir dizinin elemanı alınarak dizi oluşturulur.
- Dizinin elemanları **Arrays.sort()** metodu kullanılarak küçükten büyüğe sıralanır.

## Kullanılan Konular
- Java
- Arrays.sort() metodu
- Döngüler
- Scanner

javac Main.java
