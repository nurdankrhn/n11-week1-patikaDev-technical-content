# Harmonik Ortalama Hesaplayan Program

Bu program, dizideki sayılar için harmonik ortalamayı hesaplar. Harmonik ortalama, sayılar arasındaki dengeli bir dağılımı ölçmek için kullanılır.

## Harmonik Ortalama Hesaplama Formülü
Harmonik Ortalama formülü:
```
Harmonik Ortalama = n / (1/n1 + 1/n2 + 1/n3 + ... + 1/nn)
```
Burada:
- n: Eleman sayısı
- n1, n2, ...: Dizinin her bir elemanının harmonik serisi

## Kullanılan Konular
- Java
- Döngüler
- Diziler
