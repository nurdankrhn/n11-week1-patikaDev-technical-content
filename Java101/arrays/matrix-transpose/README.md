# Matrisin Transpozunu Bulan Program

Bu program, kullanıcıdan alınan bir matrisin transpozunu (devriğini) hesaplar ve ekrana yazdırır.

## Matrisin Transpozu Nedir?
Bir matrisin transpozu (veya devriği), matrisin satırları ile sütunlarının yer değiştirmesidir. Örneğin:
- A matrisinin transpozunu alırken, A'nın 1. satırındaki elemanlar A^T'nin 1. sütununa yerleştirilir.

## Kullanılan Konular
- Java
- Çok Boyutlu Diziler
- Döngüler
- Scanner
