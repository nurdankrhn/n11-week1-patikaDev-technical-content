# Mayın Tarlası Oyunu

Bu oyun, klasik Mayın Tarlası oyununun metin tabanlı versiyonudur. Kullanıcı, belirlediği boyutlarda bir mayın tarlasında oyunu oynar ve mayınları bulmadan tüm hücreleri açmaya çalışır.

## Özellikler:
- Kullanıcı, matris boyutlarını (satır ve sütun) girer.
- Mayın sayısı, matrisin toplam eleman sayısının çeyreği kadar rastgele yerleştirilir.
- Oyuncu, bir hücre seçerek açmaya çalışır. Eğer mayına denk gelirse oyun biter.
- Eğer oyuncu tüm mayınlar haricindeki hücreleri açarsa oyunu kazanır.

## Kullanılan Konular:
- Java
- Random sınıfı
- 2 boyutlu diziler

## Çalıştırma:

```bash
javac Main.java
java Main
