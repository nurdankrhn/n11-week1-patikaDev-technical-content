# Basit Hesap Makinesi

Bu program, kullanıcıdan iki sayı alır ve seçilen işleme göre sonucu ekrana yazdırır.

## Desteklenen İşlemler
- Toplama
- Çıkarma
- Çarpma
- Bölme

## Kullanılan Konular
- Java
- Scanner ile kullanıcıdan veri alma
- switch-case
- if-else
- Aritmetik işlemler

## Not
- Programda sayılar `int` olarak alındığı için bölme işlemi tam sayı bölmesi şeklinde yapılır.
- İkinci sayı `0` ise bölme işlemi yapılmaz ve hata mesajı verilir.

## Çalıştırma
```bash
javac Main.java
java Main
