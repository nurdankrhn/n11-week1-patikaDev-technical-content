# Login and Password Reset

Bu program, kullanıcıdan kullanıcı adı ve şifre bilgisini alır.

## Kurallar
- Kullanıcı adı `patika`
- Şifre `java123`

Eğer bilgiler doğruysa:
- `Giriş Yaptınız !`

Eğer bilgiler yanlışsa:
- Kullanıcıya şifresini sıfırlamak isteyip istemediği sorulur
- Yeni şifre eski şifre ile aynıysa:
  - `Şifre oluşturulamadı, lütfen başka şifre giriniz.`
- Farklıysa:
  - `Şifre oluşturuldu`

## Kullanılan Konular
- Java
- Scanner
- if-else
- String karşılaştırma
- equals / equalsIgnoreCase

## Çalıştırma
```bash
javac Main.java
java Main
