# Zodiac Sign Finder

Bu program, kullanıcıdan doğduğu ay ve gün bilgisini alır ve burcunu ekrana yazdırır.

## Özellikler
- Ay ve gün bilgisine göre burç hesaplanır
- Hatalı gün veya ay girilirse uyarı verilir
- Çözüm `switch-case` kullanılmadan `if-else` ile yapılmıştır

## Kullanılan Konular
- Java
- Scanner
- if / else if / else
- Ternary operator
- Koşul kontrolü

## Çalıştırma
```bash
javac Main.java
java Main
