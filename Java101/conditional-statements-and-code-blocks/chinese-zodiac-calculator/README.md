# Chinese Zodiac Calculator

Bu program, kullanıcıdan doğum yılını alır ve Çin Zodyağı burcunu hesaplayıp ekrana yazdırır.

## Hesaplama Kuralı
Doğum yılı `12` ile bölünür ve kalana göre burç belirlenir.

## Burçlar
- `0` → Maymun
- `1` → Horoz
- `2` → Köpek
- `3` → Domuz
- `4` → Fare
- `5` → Öküz
- `6` → Kaplan
- `7` → Tavşan
- `8` → Ejderha
- `9` → Yılan
- `10` → At
- `11` → Koyun

## Kullanılan Konular
- Java
- Scanner
- Mod alma işlemi (`%`)
- switch-case
