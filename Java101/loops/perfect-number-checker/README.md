# Perfect Number Checker

Bu program, kullanıcıdan bir sayı alır ve bu sayının mükemmel sayı olup olmadığını kontrol eder.

## Mükemmel Sayı Nedir?
Bir sayının kendisi hariç pozitif tam sayı çarpanları (kalansız bölen sayılar) toplamı kendisine eşit olan sayıya mükemmel sayı denir.

Örneğin:
- `6` → `1 + 2 + 3 = 6` → Mükemmel sayıdır.
- `28` → `1 + 2 + 4 + 7 + 14 = 28` → Mükemmel sayıdır.
- `1` → Mükemmel sayı değildir.

- Scanner
