# Prime Number Printer

Bu program, 1 ile 100 arasındaki asal sayıları ekrana yazdırır.

## Asal Sayı Nedir?
Asal sayı, yalnızca 1 ve kendisiyle tam bölünebilen sayıdır. Örneğin, 2, 3, 5, 7 asal sayılardır.

## Kullanılan Konular
- Java
- for döngüsü
- if koşulu
- asal sayı kontrolü

## Çalıştırma
```bash
javac Main.java
java Main
