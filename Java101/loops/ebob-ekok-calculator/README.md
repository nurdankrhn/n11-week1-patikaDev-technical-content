# EBOB and EKOK Calculator

Bu program, kullanıcıdan iki sayı alarak EBOB (En Büyük Ortak Bölen) ve EKOK (En Küçük Ortak Kat) hesaplar.

## Hesaplama Formülleri
- EBOB: İki sayının ortak bölenlerinin en büyüğü.
- EKOK: İki sayının ortak katlarının en küçüğü.
- EKOK = (n1 * n2) / EBOB

## Kullanılan Konular
- Java
- Scanner
- while döngüsü
- Method kullanımı
