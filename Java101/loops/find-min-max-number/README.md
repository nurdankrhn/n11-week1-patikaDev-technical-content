# Find Min and Max Number

Bu program, kullanıcıdan `N` tane sayı alır ve bu sayılar arasındaki en büyük ve en küçük sayıyı ekrana yazdırır.

## Özellikler
- Kullanıcıdan sayıları alır.
- En büyük ve en küçük sayıları bulur.

## Kullanılan Konular
- Java
- Scanner
- for döngüsü
- if koşulu

## Çalıştırma
```bash
javac Main.java
java Main
