# Sum of Even Multiples of Four

Bu program, kullanıcıdan sayılar alır.  
Kullanıcı tek bir sayı girene kadar giriş almaya devam eder.

Girilen sayılar arasından:
- çift olan
- ve aynı zamanda 4'ün katı olan

sayıları toplar ve sonucu ekrana yazdırır.

## Kullanılan Konular
- Java
- Scanner
- do-while döngüsü
- if koşulu
- mod alma işlemi

## Çalıştırma
```bash
javac Main.java
java Main
