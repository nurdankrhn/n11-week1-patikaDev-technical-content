# Star Diamond Pattern

Bu program, kullanıcıdan bir sayı alır ve yıldızlar ile elmas şekli oluşturur.

## Örnek

Giriş:
```text
5
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```
## Kullanılan Konular
- Java
- Scanner
- for döngüsü
- iç içe döngüler
- method kullanımı

## Not
- Girilen sayı elmasın üst yarısının yüksekliğini belirler.
- Program, önce üst kısmı sonra alt kısmı yazdırır.
