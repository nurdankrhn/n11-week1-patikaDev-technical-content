# Combination Calculator

Bu program, kullanıcıdan `n` ve `r` değerlerini alarak kombinasyon hesabı yapar.

## Kombinasyon Formülü

`C(n,r) = n! / (r! * (n-r)!)`

## Özellikler
- `n` ve `r` kullanıcıdan alınır
- faktöriyel hesabı ayrı bir metot ile yapılır
- hatalı girişler kontrol edilir
- `r > n` ise işlem yapılmaz

## Kullanılan Konular
- Java
- Scanner
- method kullanımı
- for döngüsü
- faktöriyel hesabı
- koşul kontrolü
