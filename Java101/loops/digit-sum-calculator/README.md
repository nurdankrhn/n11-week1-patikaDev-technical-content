# Digit Sum Calculator

Bu program, kullanıcıdan bir sayı alır ve sayının basamaklarının toplamını hesaplar.

## Örnek
- `1643` → `1 + 6 + 4 + 3 = 14`

## Kullanılan Konular
- Java
- Scanner
- while döngüsü
- mod alma ve bölme işlemleri
- method kullanımı

## Not
Negatif sayı girilirse hesaplama mutlak değer üzerinden yapılır.

## Çalıştırma
```bash
javac Main.java
java Main
