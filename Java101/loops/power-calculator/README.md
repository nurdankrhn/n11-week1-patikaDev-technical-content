# Power Calculator with For Loop

Bu program, kullanıcıdan bir taban ve üs değeri alır, ardından üslü sayı sonucunu `for` döngüsü ile hesaplar.

## Örnek
- 3^4 = 81
- 2^5 = 32

## Kullanılan Konular
- Java
- Scanner
- for döngüsü
- çarpma ile üslü sayı hesabı

## Çalıştırma
```bash
javac Main.java
java Main
