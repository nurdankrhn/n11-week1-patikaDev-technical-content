# ATM Project with Switch-Case

Bu program, basit bir ATM sistemini simüle eder.

## Özellikler
- Kullanıcı giriş kontrolü
- 3 hatalı giriş hakkı
- Para yatırma
- Para çekme
- Bakiye sorgulama
- Çıkış yapma

## Kullanılan Konular
- Java
- Scanner
- while / do-while
- switch-case
- method kullanımı

## Çalıştırma
```bash
javac Main.java
java Main
