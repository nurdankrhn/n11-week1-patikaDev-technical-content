# Not Ortalaması Hesaplayan Program

Bu program, kullanıcıdan 6 dersin notunu alır:
- Matematik
- Fizik
- Kimya
- Türkçe
- Tarih
- Müzik

Ardından not ortalamasını hesaplar ve:

- Ortalama 60'tan büyükse: `Sınıfı Geçti`
- Değilse: `Sınıfta Kaldı`

mesajını ekrana yazdırır.

## Kullanılan Konular
- Java
- Scanner ile kullanıcıdan veri alma
- Aritmetik işlem
- Ternary Operator

## Çalıştırma
```bash
javac Main.java
java Main
