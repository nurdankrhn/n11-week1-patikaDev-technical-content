# Dairenin Alanını ve Çevresini Hesaplayan Program

Bu program, kullanıcıdan dairenin yarıçapını ve merkez açısını alır.

Program şunları hesaplar:
- Dairenin alanı
- Dairenin çevresi
- Daire diliminin alanı

## Kullanılan Formüller

- Alan = π * r * r
- Çevre = 2 * π * r
- Daire dilimi alanı = (π * r * r * α) / 360

Bu programda π değeri **3.14** olarak alınmıştır.

## Kullanılan Konular
- Java
- Scanner ile kullanıcıdan veri alma
- Aritmetik işlemler
