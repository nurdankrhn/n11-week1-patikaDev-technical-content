# Dik Üçgende Hipotenüs ve Üçgen Alanı Hesaplama

Bu program:
- Kullanıcıdan iki dik kenarı alır
- Hipotenüsü hesaplar
- Ardından üçüncü kenarı da alarak Heron formülü ile üçgen alanını bulur

## Kullanılan Konular
- Java
- Scanner
- Math.sqrt()
- Aritmetik işlemler

## Çalıştırma
```bash
javac src/Main.java
java -cp src Main
