# Vücut Kitle İndeksi Hesaplama

Bu program, kullanıcıdan boy ve kilo bilgisini alarak vücut kitle indeksini hesaplar.

## Formül

Vücut Kitle İndeksi = Kilo / (Boy * Boy)

## Kullanılan Konular
- Java
- Scanner ile kullanıcıdan veri alma
- Aritmetik işlemler

## Çalıştırma
```bash
javac Main.java
java Main
